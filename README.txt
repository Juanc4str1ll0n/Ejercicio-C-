
Ejercicios de Listas Enlazadas en C++

Este repositorio contiene implementaciones en C++ de diferentes tipos de listas enlazadas. 
Los archivos incluidos son parte de ejercicios prácticos para comprender estructuras de datos fundamentales.

Contenido del repositorio:

- listasEnlazadas.cpp: Implementación de listas simplemente enlazadas.
- Listas_dobles.cpp: Implementación de listas doblemente enlazadas.
- Listas_doblemente_circulares.cpp: Implementación de listas doblemente enlazadas circulares.
- Ejecutables .exe para lanzamiento inmediato

Instrucciones para compilar y ejecutar:

1. Clonar el repositorio:

   git clone https://github.com/Juanc4str1ll0n/Ejercicio-C-.git

2. Navegar al directorio del proyecto:

   cd Ejercicio-C-

3. Compilar el archivo deseado con un compilador de C++:

   listasEnlazadas.cpp -o listasEnlazadas

4. Ejecutar el archivo compilado:

   ./listasEnlazadas

   Repetir los pasos 3 y 4 con los otros archivos si es necesario.

Requisitos:

- Compilador de C++ (como Visual Studio o Clion)
- Sistema operativo compatible (Windows, Linux, macOS)
